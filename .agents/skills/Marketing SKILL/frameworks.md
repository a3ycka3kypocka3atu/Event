## Стратегічні фреймворки

### Піраміда попиту (Блакитний / Червоний океан)
```
3%  — Готові купити (Червоний океан, дорогий аукціон)
17% — Збирають інформацію
20% — Усвідомлюють проблему
60% — Не усвідомлюють проблему
97% — Блакитний океан (формуєш попит сам)
```
Детальніше: [01-market-research.md](01-market-research.md)

### 5 рівнів усвідомленості (Ben Hunt / Eugene Schwartz)
| Рівень | Стан | Тип воронки |
|--------|------|-------------|
| 0 | Не знає про проблему | Контент, Advertorial |
| 1 | Знає проблему | Quiz, Advertorial |
| 2 | Знає рішення, не знає продукт | Агрегатор, VSL |
| 3 | Знає продукт, порівнює | Direct-лендінг, демо |
| 4 | Готовий купити | Checkout + offer |
Детальніше: [03-funnel-architecture.md](03-funnel-architecture.md)

### Стратегії Портера
| Стратегія | Суть | Коли |
|-----------|------|------|
| Лідерство за витратами | Найнижча ціна | Є масштаб і операційна перевага |
| Диференціювання | Унікальна цінність | Є УТП |
| Нішування | Вузький сегмент | Обмежений бюджет, є незайнятий сегмент |
| Follower (№2) | Копіювання лідера | Старт без даних і бюджету |

## Фреймворки воронок

### TOFU — MOFU — BOFU
| Рівень | Мета | Інструменти | KPI |
|--------|------|-------------|-----|
| TOFU | Обізнаність | Відео, YouTube, TikTok, SEO | Охоплення, Views |
| MOFU | Прогрів | Advertorials, квізи, лід-магніти, Email | CPL, Ліди |
| BOFU | Конверсія | Ретаргетинг, покинуті кошики, оффери | CPA, ROAS |

### STDC (See-Think-Do-Care) — Google
| Етап | Контент | Канали | KPI |
|------|---------|--------|-----|
| See | Безкоштовна цінність | Блог, YouTube, Social | Охоплення |
| Think | Переваги, майстер-класи | Контент, ретаргет | CPL |
| Do | Прямий офер | Paid Search, інфлюенсери | CPA |
| Care | Утримання | Email, Push, ком'юніті | LTV, Churn |

### AARRR (Pirate Metrics — Dave McClure)
| Етап | Опис | Бенчмарк |
|------|------|----------|
| Acquisition | Залучення | CPI, CTR |
| Activation | Перша дія | Відвал 30-60% від інсталу |
| Retention | Повернення | Без нього немає LTV |
| Revenue | Оплата | ~2% з інсталу |
| Referral | Віральність | Запрошення друзів |

### Value Ladder (Russell Brunson)
```
Lead Magnet (безкоштовно) → Tripwire ($5-47) → Core Offer ($97-997) → Profit Maximizer ($997-5000) → High-Ticket ($5000+)
```

## Фреймворки оферу і аудиторії

### Value Equation (Alex Hormozi)
```
           Результат мрії × Ймовірність досягнення
Цінність = ──────────────────────────────────────────
           Час до результату × Зусилля і жертви
```
Збільшуй чисельник (результат, докази) і зменшуй знаменник (час, складність).

### Grand Slam Offer (Hormozi)
Офер настільки сильний, що відмова = дурість. Стекінг: продукт + бонус 1 + бонус 2 + бонус 3 + гарантія. Загальна цінність $XXX → ціна $YY.

### ICP + Кути комунікації (Антонов)
```
3 головні болі × 5-6 результатів = 10+ кутів комунікації
Кожен кут = окрема рекламна гіпотеза
```

### ДНК Клієнта — 5-частний аналіз (Yasko)
1. Ідентичність, цінності, мировоззрение
2. Болі, страхи, тригери
3. Бар'єри, міфи, минулий досвід
4. Трансформація, заперечення ринку
5. Тригери покупки, механіка впливу

Повна методологія: [client-dna/SKILL.md](../client-dna/SKILL.md)

### Матриця сегментів
```
1 сегмент → 1 біль → 1 ДНК → 1 офер
3 сегменти × 3 болі = 9 мікро-ДНК + 9 офферів
```

## Креативні фреймворки

### Hook — Story — CTA (Відеокреатив)
```
Hook (0-3 сек): Зупинити скролінг
Story: Довести цінність (PAS, Before/After, Tutorial)
CTA (останні 3-5 сек): Заклик до дії
```

### Формули наративу
| Формула | Структура |
|---------|-----------|
| **PAS** | Problem → Agitation → Solution |
| **AIDA** | Attention → Interest → Desire → Action |
| **BAB** | Before → After → Bridge |

### Піраміда креативів
```
New (нова концепція) → Variation (1-3 зміни) → Refresh (3+ змін, новий layout)
70% часу — на Variations успішних креативів
3-5 варіацій на концепт
~3% з 100 тестів стають топ-перформерами
```

### Метод Діснея (Брейншторм)
```
Мрійники (без критики) → Реалісти (як зробити) → Критики (ризики і Policy)
```

## Фінансові формули

### Ключові метрики
```
CAC = Ad Spend / Customers
LTV = Total Revenue from Customer over Lifetime
ROAS = Revenue / Ad Spend
Break-even ROAS = 1 / Margin
CPA = CPC / CVR
ROAS (розрахунковий) = AOV / ((CPM / 1000 / CTR) / CVR)
Payback Period = CAC / Monthly Revenue per Customer
```

### Золоте співвідношення
```
CAC : LTV = 1 : 3
```

### Ціна продукту → Тип воронки
| Ціна | Воронка | CAC ліміт |
|------|---------|-----------|
| $5-50 | Quiz, Advertorial, Tripwire | $5-15 |
| $50-300 | VSL, вебінар, лід-магніт | $15-80 |
| $300-1000 | Діагностичний дзвінок | $80-300 |
| $1000+ | Ecosystem (YouTube, Telegram, дзвінок) | $150-500+ |

## Аналітичні фреймворки

### Bottleneck-аналіз воронки
```
Impressions → CTR → Clicks → LP View Rate → CVR → Purchase → Upsell → Revenue
Де найбільше падіння = bottleneck = фокус оптимізації
```

### Діагностика проблеми
| Симптом | Діагноз |
|---------|---------|
| Високий CPM + низький CTR | Вигорання креативу |
| Високий CTR + 0 конверсій | Клікбейт або зламаний лендінг |
| CTR ок + покупок нема | Проблема воронки/ціни |
| CPA росте через 2-3 дні | Вузька аудиторія |

### Чекліст масштабування
```
✅ True ROAS > Break-even ROAS
✅ CAC:LTV = 1:3+
✅ Payback Period < 12 міс
✅ 100+ конверсій
✅ Стабільність 2-4 тижні
```
