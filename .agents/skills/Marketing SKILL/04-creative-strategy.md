# Креативна стратегія та продакшн — Повний довідник

Цей документ — повна база знань зі створення рекламних креативів: від ресерчу і генерації ідей до продакшну, тестування і масштабування. Використовуй його коли потрібно розробити креативну стратегію, написати ТЗ на креатив, оцінити ефективність або побороти вигорання.

---

## 1. Ресерч і пошук ідей

### 1.1. Spy-сервіси — що шукати і як аналізувати

**Facebook Ad Library / TikTok Creative Center (Top Ads):**
- Шукай креативи з високою залученістю (500,000+ лайків) та високим бюджетом — це означає, що реклама окупається
- Аналізуй метрики CTR та CVR. Графіки покажуть, на якій секунді люди клікають або відвалюються (аналіз хуків)
- Для країн ЄС (EU Transparency Law): в Ad Library та TikTok Library видно точні дані по охопленню (Reach) та приблизні витрати (Spend), таргетинг за віком і гео. Знаючи середній CPM, можна вирахувати реальні бюджети конкурентів

**Головне правило ресерчу:** Шукай не просто креатив, а **зв'язку (Креатив + Воронка)**. Завжди клікай на рекламу конкурента, щоб побачити куди вона веде (квіз, стаття, магазин). Зв'язка важливіша за окремий креатив.

**Додаткові інструменти:**
- **BigSpy / PiPiAds:** Креативи зі всіх платформ (FB, TikTok, Pinterest, YouTube) з даними про кількість днів відкруту та показів
- **Sensor Tower / AppMagic:** Для мобільних додатків і ігор. Фільтрація топ-креативів конкурента за період і гео
- **Відгуки і коментарі:** Trustpilot, Reddit, AppStore, коментарі під рекламою конкурентів — джерело болей на мові клієнта
- **TikTok тренди:** Органіка TikTok, хештеги (#TikTokMadeMeBuyIt, #AmazonFinds), комерційна бібліотека музики (треки, що набирають популярність)

### 1.2. Метод брейншторму Діснея (Disney Creative Strategy)

Три ізольовані ролі для уникнення конфліктів і блокування ідей:

**Етап 1 — Мрійники (Dreamers):**
- Мета: максимальна кількість ідей, навіть абсурдних
- Головне правило: ЖОДНОЇ критики. "Кількість породжує якість"
- Запитання: "Що якби ми могли зробити що завгодно без обмежень?"

**Етап 2 — Реалісти (Realists):**
- Беруть ідеї мрійників і думають як їх реалізувати
- Які інструменти потрібні? Де знайти акторів? Який бюджет? Покроковий план

**Етап 3 — Критики (Critics):**
- Оцінюють ризики: чи не порушить Policy платформи? Чи вистачить бюджету?
- На роль критиків найкраще садити SEO або CMO — вони розуміють специфіку бізнесу та обмеження

### 1.3. Від ДНК Клієнта до креативних ідей

Робота починається не з дизайну, а з портрета клієнта:

```
3 головні болі клієнта
        ×
5-6 результатів продукту
        =
10+ кутів комунікації
        ↓
Кожен кут = основа для хука у креативі
```

Кожен кут комунікації — це окрема гіпотеза для тестування. Один кут = один набір креативів (3-5 варіацій).

## 2. Структура креативів

### 2.1. Формула відеокреативу: Hook → Story → CTA

Оптимальна довжина: **15-23 секунди** (для TikTok 15-20 сек).

**HOOK (0-3 секунди) — Зупинити скролінг**

Якщо юзер не зупинився за 3 секунди, решта відео не має значення.

**Типи хуків:**

| Тип | Опис | Приклад |
|-----|------|---------|
| Візуальний | Неон, зміна кадру, динамічний фон, незвичне дійство | Людина кидає телефон у воду |
| Текстовий-питання | Питання, що зачіпає біль | "Чому ти досі не можеш схуднути після 40?" |
| Текстовий-ствердження | Шокуюче або контрінтуїтивне твердження | "Біг не допомагає худнути" |
| Руйнування міфу | Заперечення популярного переконання | "Забудьте про калорії — ось що насправді важливо" |
| Новинний | Стиль жовтої преси, breaking news | "Колишній співробітник Facebook порушив мовчання..." |
| Соціальний доказ | Результат або кейс | "Як Анна заробила перші $10k за 30 днів" |
| POV | Ламання стереотипів від першої особи | "POV: ти купив годинник за $30, а всі думають він за $30,000" |
| Тренд | Використання трендового звуку або формату | Адаптація вірального мему під продукт |
| Провокація | Виклик або протиріччя | "Ваш маркетолог вас обманює" |

**STORY (основна частина) — Довести цінність**

Сюжет, який долає бар'єри та упередження. Формати наративу:
- **Problem-Agitation-Solution (PAS):** Покажи проблему → Підсиль біль → Дай рішення
- **Before/After:** Життя з проблемою → Продукт → Життя без проблеми
- **Tutorial:** Покроковий гайд використання (високий рівень довіри)
- **Testimony:** Відгук реальної людини або UGC
- **Showcase:** Швидка нарізка товарів (для E-com з широким асортиментом)

**CTA (останні 3-5 секунд) — Заклик до дії**

- Візуальні ефекти: затемнення екрану, блимання кнопки
- Лайфхак: використовуй ДВА CTA — один у тексті креативу (емоційний: "Стань найцікавішою людиною в кімнаті"), другий на кнопці (прямий: "Buy Now" / "Learn More")
- CTA має бути конкретним: не "дізнайся більше", а "отримай свій план за 3 хвилини"

### 2.2. Формули для скриптів креативів

**PAS (Problem → Agitation → Solution):**
```
[Hook] "Втомився від того, що реклама не працює?"
[Problem] "Ти витрачаєш $500 на рекламу і отримуєш 3 ліди..."
[Agitation] "А твій конкурент з гіршим продуктом вже забирає твоїх клієнтів..."
[Solution] "Ось система, яка змінила все для 3000 підприємців..."
[CTA] "Скачай безкоштовний гайд — лінк в описі"
```

**AIDA (Attention → Interest → Desire → Action):**
```
[Attention/Hook] Шокуюче твердження або візуальний стоп
[Interest] Цікавий факт або історія
[Desire] Демонстрація результату / трансформація
[Action] Конкретний CTA
```

**BAB (Before → After → Bridge):**
```
[Before] Покажи поточний стан (біль)
[After] Покажи бажаний стан (мрія)
[Bridge] Твій продукт — міст між ними
```

### 2.3. Структура статичних банерів

Статика читається згори донизу, від великого до малого:

```
1. Очі / Обличчя героя (притягує увагу)
2. Великий заголовок — Хук
3. Основний контент — візуалізація товару / результату
4. Дрібний текст — умови, деталі
5. CTA-кнопка
```

**Правила:**
- Персонаж дивиться ПРЯМО в камеру (працює краще, ніж відведений погляд)
- Уникай перевантаження текстом
- Замість реальних людей часто працює вектор або поп-арт (особливо для дейтингу, делікатних ніш)
- Розмір: 4×5 для стрічки Facebook/Instagram, 9:16 для Stories/Reels

## 3. Піраміда класифікації креативів

Для масштабування: **70% часу — на варіації успішних креативів**, а не на нові концепції.

### 3.1. Три рівні

| Рівень | Що це | Що змінюється | Приклад |
|--------|-------|---------------|---------|
| **New** | Абсолютно нова концепція | Все | Нова ідея, новий формат, новий кут |
| **Variation** | Мінорні зміни (1-3 елементи) | Заголовок, колір, музика, CTA | Змінили текст хука з питання на ствердження |
| **Refresh** | Суттєві зміни (3+ елементи) або зміна layout | Композиція, герої, стилістика | Нові герої + новий layout при збереженні суті |

### 3.2. Як масштабувати через варіації

1. Візьми успішний креатив (віннер)
2. Розклади на блоки (mind map): хук, кольори, музика, герой, CTA, текст
3. Визнач що саме дає результат (напр., дівчина + персиковий фон = покупки, фіолетовий фон = високий CTR)
4. Комбінуй: дівчина + фіолетовий фон = варіація для тесту
5. На один концепт — **3-5 варіацій**

### 3.3. Статистика ефективності

- Із 100 протестованих креативів лише **~3% стають топ-перформерами**
- Це нормально — тому потрібний конвеєр, а не пошук "одного ідеального" креативу
- 15-20% бюджету кампанії — на тестування нових креативів (5% якщо бюджет >$1M)

## 4. Продакшн

### 4.1. Написання ТЗ (технічне завдання)

ТЗ має бути настільки детальним, щоб не залишалось фрази "зроби гарно".

**Для статики:**
- Розмір (4×5, 1:1, 9:16)
- Точний текст заголовку
- Опис візуальних елементів з референсами/лінками на стоки
- Кольорова гама (HEX-коди)
- Фон та форма CTA-кнопки

**Для відео:**
- Розмір (9:16 або 4:5), таймінг (15-20 сек)
- Детальна розкадровка:
  - "Перші 3 сек — білий екран і текст [Текст]"
  - "Далі відео [Лінк], де дівчина малює"
  - "Додай ефект переходу (скручування)"
  - "Наклади фонову музику [Лінк]"
  - "На 10-й секунді закадровий голос [Аудіофайл] каже [Текст]"

### 4.2. UGC-кріейтори — де шукати

| Платформа | Ціна | Особливість |
|-----------|------|-------------|
| **Fiverr / Upwork** | $50-100 за креатив | Дешевше, простіше, масова генерація |
| **TikTok Creator Marketplace** | Дорожче | Офіційна біржа, перевірені кріейтори |
| **TikTok Creative Challenge (TTCC)** | Безкоштовно | Кріейтори роблять креативи за бриф, умова — витратити від $200 на рекламу вдалого. Переважно для США |

### 4.3. Spark Ads (TikTok)

Реклама від імені органічного профілю або кріейтора. Клік на CTA веде на сайт/додаток, клік на аватарку або свайп — в органічний профіль (+ підписники).

**Застереження:** Використовуй режим "Тільки для реклами" (Ads-only mode), щоб жорсткі перформанс-креативи не публікувались в органічній стрічці бренду і не призвели до бану через різницю в Policy.

### 4.4. Кількість креативів для тестового запуску

- В одну рекламну групу (Ad Group): **3-5 креативів**
- Бюджет на тест групи: **5-10× від цільового CPA** (або 2× від LTV)
- Приклад: CPA $10 → бюджет на тест від $50
- Для автоматизованих кампаній (Smart Performance Campaign): **40+ асетів** (напр., 10 відео × 5 текстів)

## 5. Вигорання креативів (Creative Fatigue)

### 5.1. Що вигорає і чому

Вигорає не сам креатив, а **рекламна група (Ad Group)**. Алгоритм песимізує групу, коли вона вичерпує початковий пул аудиторії.

**Ознаки вигорання:**
- Перші 1-4 дні CPA відмінний, потім різко зростає (×2) і тримається 2-3 дні
- Падає CTR
- Зростає Frequency (частота показу одній людині)

### 5.2. Життєвий цикл

У TikTok (Web + Android) середній цикл прибуткової групи — **5-7 днів**. У Meta зазвичай довший, але принцип той самий.

### 5.3. Стратегії боротьби

**Перезапуск:**
- CPA > Target протягом 2 днів → вимикай групу
- Не чекай завершення Learning Phase, якщо старт одразу дорогий

**Ротація і дублювання:**
- НЕ роби чисту копію — система бачить і песимізує
- Зміни мінімум один параметр: таргетинг (Broad → Інтереси), текстову підводку, або один креатив у групі
- Для алгоритму це "нова" кампанія з новим пулом аудиторії

**Таймінг перезапусків:**
- Найкращий час для масових перезапусків: **четвер-субота** (на вихідних більше трафіку)

**Воскресіння (Resurrection):**
- Успішні групи, які вигоріли і були вимкнені, можна увімкнути через **2-3 тижні**
- Часто знову показують круті результати завдяки історичній даті

## 6. Кейси

### Кейс "Креатив на $53,000" (Павло Антонов)

- **Задача:** Залучити холодну аудиторію в безкоштовну лід-воронку (міні-курс з маркетингу) → апсейл на платне навчання
- **Підхід:** Замість "Безкоштовний курс з маркетингу" — новинний підхід (News-style), натхненний обкладинками жовтої преси
- **Хук:** "Колишній співробітник Facebook порушив мовчання і віддає секретну інформацію (про закупівлю трафіку)"
- **Результат:** Message Match з лендінгом через Unbounce + Zapier. Зібрано 650+ анкет → $53,000 доходу з однієї зв'язки

### Кейс Draw Online (Genesis)

- Гра: намалювати фігуру однією лінією
- Інсайт: в коментарях юзери писали "це ж легко" і кидали скріни
- Креатив: гравець навмисно тупить і не може вирішити просту задачу → шквал залученості (ефект "я можу краще!")

### Кейс адаптації трендів (Барбі/Оппенгеймер)

- Школа Squat адаптувала тренд через гівки/меми
- Genesis використали моушн-дизайн у стилі тренду для ігор
- Принцип: бери вірусний формат → адаптуй під свій продукт

### Бенчмарки

| Метрика | Значення | Контекст |
|---------|----------|----------|
| CTR відео (TikTok/Meta Broad) | 1.0-2.0% | Хороший/нормальний |
| CTR статика (Web, історичний) | >11% | Рідкісний космос |
| Бюджет на тест креативів | 15-20% від загального | 5% якщо бюджет >$1M |
| Топ-перформери з тестів | ~3% зі 100 | Нормальна конверсія |
| Варіації на концепт | 3-5 штук | Мінімум для тесту |
| Креативів в Ad Group | 3-5 | Стандарт |

## 8. Зв'язки з іншими модулями

- **ДНК Клієнта → Креатив:** Болі, страхи і мова клієнта стають хуками. 3 болі × 5-6 результатів = 10+ кутів = 10+ гіпотез для креативів.
- **Воронка → Креатив:** "Ми шукаємо не креатив, ми шукаємо ЗВ'ЯЗКУ". Креатив і лендінг — єдина система. Message Match = обіцянка в рекламі = перший екран лендінгу.
- **Юніт-економіка → Тестування:** Бюджет на тест = 5-10× від CPA. Креатив з низьким CTR (0.8%), але який дає конверсії в межах CPA — НЕ вбивати.
- **Аналітика → Оптимізація:** Дашборди показують яка зв'язка працює, на якій секунді відвалюються, які хуки дають найкращий Thumbstop Rate.
